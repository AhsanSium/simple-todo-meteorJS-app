# simple-todo-meteorJS-app

## Features

    1. User Can Login.

    2. User Can Add New Task.

    3. User Can Change Task to Pending.

    4. User Can Delete Task After Completed.

    5. User Data is Protected.

    6. User Data is filtered.

## Technologies

    1. Meteor.js

    2. React

    3. MongoDB

    4. Meteor Cloud
